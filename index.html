Pyzza Bytes

FOOD ORDERING APP

  • HTML5
  • CSS3
  • Vanilla JavaScript
  • Bootstrap

Pyzza Bytes is an food ordering app that allows the user to choose from a wide range of food. User can choose how they would like their order by using the dropdown options. This was a learning project to get a better grasp on the fundementals of JavaScript and improving my proficiency utilizing bootstrap.

Project Overview

Why did I build this project?

I built this project for learning experience. I wanted to make a challenging project that would allow me to continously add more features to it as my skills improve. This project allows me to get as close to a real world project as possible. I learned so much after implementing a few of the features into this project and I'm excited to see where it takes me a few months from now.

Build Difficulties?

Implementing a nested modal window to display the current order. This became our first real challenge as I was still getting familiar with utilizing bootstrap components. It took a bit of time to think about how I wanted to add the order list and a second modal window to display the order submission. Luckily, the bootstrap documentation is very thorough and easy to read, I was able to come up with a clean layout.

Future Features:

  • Allow user to remove item from order
  • Discount for certain items based on current day of the week
  • Create an account

Redline Detailing

APPOINTMENT SCHEDULING APP

  • HTML5
  • CSS3
  • Vanilla JavaScript

This was a learning project that sharpened my CSS skills while also introducing JavaScript functionality with mobile navigation. All responsive web design was done via media queries by hand that really helped me learn what it takes to create a responsive website. The process was challenging in the beginning but I eventually learned how they operate and it wasn't nearly as difficult as it seemed beforehand. It felt very rewarding being able to create a responsive design by hand without relying on bootstrap. This will be a project I will revisit to keep adding on more features to hone my JavaScript skills.

Project Overview

Why did I build this project?

This was a learning project to get additional practice with CSS. After creating my previous site(Echo Trails), I was able to focus more on the advanced CSS layouts by utilizing grid and flexbox for my container content. These two tools together made creating layouts effortless. I was also able to create a unique design utilizing shadows that really made the package section stand out.

Build Difficulties?

Having my hero image and other site images look good became a bit of a challenge in the beginning but I realized my mistake and by adjusting the dimensions into viewport width and height, solved my problem.

Future Features:

  • Cancel an appointment
  • Update an existing appointment
  • View current appointment
  • Create an account

Echo Trails

OUTDOOR ADVENTURE APP

  • HTML5
  • CSS3

Echo Trails is a one-stop adventure site that bundles up exciting activities in one place. The site was built with HTML and CSS. This was a learning project that allowed me to obtain a strong grasp on HTML and CSS before moving on to JavaScript.

Project Overview

Why did I build this project?

I built this project to put what I learned into practice by putting away tutorials and testing my knowledge on the topics covered. Being my first page, I spent a good amount of time reading documentation as I was going through the process of creating each section.

Build Difficulties?

Learning flexbox and grid were a bit challenging to grasp in the beginning. With the use of MDN docs, they started to make sense and I came to love utilizing these great tools for my site layouts.

Media Queries were challenging to do by hand. I had to read documentation to better understand them and once I implemented my media query designed for desktop, it all started to come together. I enjoyed learning how media queries operate behind the scenes.

Future Features:

  • Add functionality to purchase a package
  • Add functionality to create an account
  • Add mobile navigation menu layout
